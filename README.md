## 🚀 How to Use / Run

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/your-repo-name.git
   ```
2. **Open in browser:**
   - Open `index.html` in any web browser.
   - Modify the script file path in `index.html` (e.g., `<script src="script9.js"></script>`) to test different topics.
   - Open your browser's Developer Tools (`F12` or right-click -> **Inspect**) and navigate to the **Console** tab to view the outputs!

3. **Run via Node.js (Optional):**
   If you have Node.js installed, you can execute any script directly in your terminal:
   ```bash
   node script9.js
   ```
